# Data Policy

This repository contains small sampled JSONL files under `dataset/` for convenience.

Please make sure your use complies with each original dataset's license and terms.
You are responsible for checking redistribution restrictions before publishing derived
artifacts.

Datasets referenced by this project include (but are not limited to):

- LMSYS Chat
- GSM8K
- Alpaca
- MMLU
- SQuAD

For large-scale experiments, we recommend downloading datasets from official sources
and regenerating local samples using `dataset/get_dataset.py`.
